# Quản lý Quán Cà Phê

## Giới thiệu

Đây là tiểu luận môn lập trình trên Windows được thực hiện bởi **Trần Ngọc Minh**. Ứng dụng này là một hệ thống quản lý cho quán cà phê, giúp quản lý các hoạt động hàng ngày của quán một cách hiệu quả.

## Tính năng

- Đăng nhập với tài khoản mặc định: 
  - Tài khoản: `admin`
  - Mật khẩu: `0`
  - Mật khẩu được mã hoá dưới database để bảo mật thông tin.
- Giao diện hiển thị chuẩn và đầy đủ nhất ở scale 100% của Windows.
- Các chức năng quản lý cơ bản:
  - Quản lý danh sách menu cà phê.
  - Quản lý danh sách đơn hàng.
  - Thống kê doanh thu.
  - Quản lý nhân viên.
  - ...

## Yêu cầu hệ thống

- Hệ điều hành: Windows.
- Thực hiện tốt nhất ở scale 100% của Windows.
